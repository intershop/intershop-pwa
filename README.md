# Intershop Progressive Web App

Welcome to the Intershop PWA project!

Hallo Welt

The Intershop PWA is an Angular-based progressive web app storefront for the Intershop Commerce Platform.</br>
Accompany us on a journey for great cutting-edge e-commerce and take the chance to make it your journey, too.

If you want to get a first impression, please visit our [public demo](https://intershoppwa.azurewebsites.net/home).

More information on the PWA can be found [here](https://www.intershop.com/en/progressive-web-app).

We welcome contributions from the community! <br/>
To contribute, please have a look at our [Contribution Guidelines](./CONTRIBUTING.md)
